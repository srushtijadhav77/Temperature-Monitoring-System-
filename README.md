# Temperature Monitoring System

## Project Overview
This project is an Arduino-based Temperature Monitoring System that measures temperature using the TMP36 sensor and displays it on a 16x2 LCD. The system uses LEDs and a buzzer to indicate low, normal, and high temperature conditions.

## Features
- Real-time temperature monitoring
- 16x2 LCD display
- Green LED for normal temperature
- Yellow LED for low temperature
- Red LED for high temperature
- Buzzer alert for high temperature

## Components Used
- Arduino Uno
- TMP36 Temperature Sensor
- 16x2 LCD
- Breadboard
- LEDs
- Buzzer
- Resistors
- Jumper Wires
